# Project OU

A full-stack web application providing Ontario high school students with comprehensive university admissions data, resources, and tools.

## Tech Stack

- **Framework:** Next.js 14 (App Router, Server Actions)
- **Auth:** Clerk
- **Database:** MongoDB with Prisma ORM
- **Caching:** Redis
- **AI/ML:** OpenAI embeddings, Pinecone vector database
- **Storage:** AWS S3
- **Styling:** Tailwind CSS, Radix UI

## Features

- **Admissions Data Browser** — Searchable, filterable table of real admission decisions with program averages and statistics
- **Analytics Dashboard** — Visualizations of admission trends, acceptance rates, and grade distributions using Recharts
- **Chance Me Calculator** — AI-powered admission probability estimator based on historical data
- **Community Tips** — User-contributed advice with upvoting, attachments (S3), and semantic search (Pinecone)
- **AI Chatbot** — Context-aware assistant using RAG to answer admissions questions
- **Program Comparison** — Side-by-side comparison of universities and programs
- **GPA Calculator** — Ontario high school average calculator
